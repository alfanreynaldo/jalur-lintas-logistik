================================================================
  PANDUAN FOTO — PT Jalur Lintas Logistik
================================================================

Simpan semua foto Anda di folder ini (images/).
Setelah foto disimpan, buka index.html dan ikuti petunjuk
komentar <!-- --> di setiap bagian untuk mengaktifkan foto.

----------------------------------------------------------------
DAFTAR FOTO YANG DIBUTUHKAN
----------------------------------------------------------------

[ HERO — Tampilan Utama Website ]
  Nama file : hero-kereta.jpg
  Ukuran    : min. 1200 x 800 px (landscape)
  Isi foto  : Foto kereta kargo / lokomotif tampak samping
  Cara pakai: Buka index.html, cari komentar "FOTO HERO",
              hapus <!-- dan --> untuk mengaktifkan tag <img>

[ TENTANG KAMI ]
  Nama file : tentang-kami.jpg
  Ukuran    : min. 800 x 600 px (landscape)
  Isi foto  : Foto kantor, tim, atau fasilitas perusahaan
  Cara pakai: Cari komentar "FOTO TENTANG KAMI" di index.html

[ GALERI OPERASIONAL ]
  Nama file : galeri-1.jpg  → Foto utama (tampil besar) — kereta/armada
  Nama file : galeri-2.jpg  → Bongkar muat kontainer
  Nama file : galeri-3.jpg  → Stasiun / depo operasional
  Nama file : galeri-4.jpg  → Tim / karyawan
  Nama file : galeri-5.jpg  → Gudang / fasilitas logistik
  Ukuran    : min. 800 x 600 px per foto

[ LAYANAN (opsional — jika ingin foto di kartu layanan) ]
  Nama file : layanan-kargo-kereta.jpg
  Nama file : layanan-kontainer.jpg
  Nama file : layanan-industri.jpg
  Nama file : layanan-reefer.jpg
  Nama file : layanan-lastmile.jpg
  Nama file : layanan-tracking.jpg
  Ukuran    : min. 600 x 400 px per foto

[ LOGO KLIEN (opsional — jika ingin logo perusahaan klien) ]
  Nama file : logo-semen-indonesia.png
  Nama file : logo-pupuk-indonesia.png
  Nama file : logo-krakatau-steel.png
  Nama file : logo-bukit-asam.png
  ... dst sesuai klien
  Format    : PNG dengan background transparan (.png)
  Ukuran    : min. 200 x 80 px

----------------------------------------------------------------
FORMAT FOTO YANG DIDUKUNG
----------------------------------------------------------------
  .jpg / .jpeg   → Direkomendasikan untuk foto
  .png           → Direkomendasikan untuk logo (transparan)
  .webp          → Format modern, ukuran lebih kecil

----------------------------------------------------------------
TIPS
----------------------------------------------------------------
  • Kompres foto sebelum diupload agar website lebih cepat.
    Gunakan: https://squoosh.app atau https://tinypng.com
  • Ukuran ideal per foto: di bawah 500 KB
  • Gunakan nama file tanpa spasi (pakai tanda hubung -)
